## 1 tuple_count.py:- is a tuples that Counts the number of occurrences of a value.
# example:-
clothes = ("pants", "tshirt", "sweater". "dress") # a variable is specified in a list
print(clothes.count("sweater")) # "2" is expected to be printed

## 2 tuple_index.py:- is a tuples that Returns the index of the first occurrence of a
 value.
 # example:-
clothes = ("pants", "tshirt", "sweater". "dress") # a variable is specified in a list
print(clothes.index("sweater")) # "2" is expected to be printed

## 3 tuple_concatenation:- is a tuples that You can combine two or more tuples using the + operator.
# example:-
odd_numbers = (1, 3, 5)
even_numbers = (2, 4, 6)
combined = odd_numbers + even_numbers
print(combined)  # Output: (1, 3, 5, 2, 4, 6)
